Homework JavaScript 101 : 10 Function


(1) .push()

push() เป็น function ที่เพิ่มค่าเข้าไปในตัวแปร array โดยต่อท้ายจากค่าเดิม แล้วคืนค่าจำนวนรายการทั้งหมดของตัวแปร array กลับมา ซึ่งมีผลให้ตัวแปร array มีการเปลี่ยนแปลง
Example

{{ num }}



(2) .pop()

pop() ลบค่าตัวสุดท้ายออกจากตัวแปร array แล้วคืนค่านั้นกลับมา มีผลให้ตัวแปร array มีการเปลี่ยนแปลง
Example

{{ num }}


(3) .shift()

shift() ลบค่าตัวแรกออกจากตัวแปร array แล้วคืนค่านั้นกลับมา มีผลให้ตัวแปร array มีการเปลี่ยนแปลง
Example

{{ num }}


(4) .unshift()

unshift() เพิ่มค่าเข้าไปในตัวแปร array โดยเพิ่มเข้าไปเป็นตัวแรก คืนค่าเป็นจำนวน array ทั้งหมด มีผลให้ตัวแปร array มีการเปลี่ยนแปลง
Example

{{ num }}



(5) .concat()

concat() ใช้สำหรับสร้างตัวแปร array ใหม่จาก การรวมตัวแปร array ตั้งแต่ 2 ตัวขึ้นไป โดยที่ตัวแปร array ที่นำมา รวมกันนั้น ยังเหมือนเดิม ไม่มีการเปลี่ยนแปลงใดๆ
Example

menbers = [1 , 2 , 3]

menberspop = [4 , 5 , 6]

result = this.menbers.concat(this.menberspop);


result:

{{ result }}


(6) .join()

join() การนำค่าแต่ละค่าในตัวแปร array มารวมกันเป็นข้อความ และส่งค่ากลับเป็นข้อความ ที่มีตัวคั่นค่าตัวแปรแต่ละค่า ตามที่กำหนด ถ้าไม่ได้กำหนด จะเป็นเครื่องหมาย comma (,) ให้โดยอัตโนมัติ
Example

cars=["Toyota","Honda","Isuzu"];


[1] carsUseJoin:string = this.cars.join()

กรณีไม่กำหนดตัวคั่นค่าตัวแปร => {{ carsUseJoin }}


[2] carsFixJoin:string = this.cars.join(' : ')

กรณีกำหนดตัวคั่นค่าตัวแปร => {{ carsFixJoin }}


(7) .reverse()

reverse() ใช้สำหรับ กลับลำดับค่าของตัวแปร array เช่น ค่าจากเดิม คือ 1 2 3 เมื่อใช้ reverse() ลำดับค่าใหม่จะเท่ากับ 3 2 1 มีผลให้ตัวแปร array มีการเปลี่ยนแปลง
Example

{{ num }}


(8) .length

length คำสั่ง หรือฟังก์ชันสำหรับนับจำนวน เช่น นับจำนวน Array หรือนับจำนวนตัวอักษรจากข้อความ โดยคำสั่ง length จะคืนค่าเป็นตัวเลขจำนวนเต็มบวกที่สามารถนับได้
Example
{{ 'Length in ' + '[ ' + text1 + ' ]' + ' is ' + sumLengthcars}}

(9) .find()

find() ใช้สำหรับหาค่าใน array ซึ่งถ้าค่าตรงกับฟังก์ชันก็จะคืนค่านั้นออกมา หากไม่มีค่าใดที่ตรงตามฟังก์ชันก็จะคืนค่าเป็น undefined
Example

cars=["Toyota","Honda","Isuzu"];


findcar = this.cars.find(element => element === "Honda");


result:

{{ findcar }}


(10) .filter()

filter() ใช้สำหรับสร้าง Array ใหม่ เมื่อผ่านเงื่อนไขที่กำหนด จาก Array ที่ใช้เรียก.

includes() กำหนดว่า Array มีค่าบางอย่างในรายการนะ

Example

menu = ['Espresso', 'Iced Coffee', 'Cappuccino', 'Latte', 'Iced Tea', 'Iced Pink Milk'];


resultmenu = this.menu.filter(element => element.includes('Iced'))


result:

{{ resultmenu }}
